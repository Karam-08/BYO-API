Welcome to My Recipe API

This API allows you to manage recipes and tags. Below are the available endpoints and how to use them.

Overall there are 25 recipes and 10 tags.

Recipe ID's: r1, r2, r3... r25

Primary Resource: /recipes

  • POST /recipes - Creates a new recipe
  • GET /recipes - List every recipe
  • GET /recipes/:id - Gets a single recipe
  • PATCH /recipes/:id - Updates a recipe
  • DELETE /recipes/:id - Deletes a recipe

Secondary Resource: /tags

  • GET /tags - List all tags

API endpoints:

  • View every recipe:

    localhost:5000/recipes


  • View a single recipe:

    localhost:5000/recipes/:id

    (replace :id with the actual recipe ID)


  • View a recipe's ingredients:

    localhost:5000/recipes/:id/ingredients

    (replace :id with the actual recipe ID)


  • View a recipe's rating:

    localhost:5000/recipes/:id/rating

    (replace :id with the actual recipe ID)


  • View every tag:

    localhost:5000/tags

Example POST Request

POST /recipes
Content-Type: application/json

{
    "dishName": "Chocolate Cake",
    "ingredients": "flour, sugar, cocoa powder, eggs, butter, milk",
    "tags": ["dessert"]
    "rating": 9
}

Example Response

{
    "id": 1,
    "dishName": "Chocolate Cake",
    "ingredients": "flour, sugar, cocoa powder, eggs, butter, milk",
    "tags": ["dessert"]
    "rating": 9
}
