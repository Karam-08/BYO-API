Welcome to My Recipe API

This API allows you to manage recipes and tags. Below are the available endpoints and how to use them.

Primary Resource: /recipes

  • POST /recipes - Creates a new recipe
  • GET /recipes - List every recipe
  • GET /recipes/:id - Gets a single recipe
  • PATCH /recipes/:id - Updates a recipe
  • DELETE /recipes/:id - Deletes a recipe

Secondary Resource: /tags

  • GET /tags - List all tags

Viewing the API:

  • /api/recipes - Views every recipe
  • /api/recipes/ID - Views a specific recipe from the ID
  • /api/tags - Views all tags
  • Note: You must add the api link to the end of the URL to view the API things
  • EX: http://localhost:5000/api/recipes

Example POST Request

POST /recipes
Content-Type: application/json

{
    "name": "Chocolate Cake",
    "tags": ["dessert", "baking"]
}

Example Response

{
    "id": 1,
    "name": "Chocolate Cake",
    "tags": ["dessert", "baking"],
    "createdAt": "2025-10-09T18:00:00Z"
}
